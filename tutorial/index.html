PPV Stream - Page Guide

Explains the function of every page for both users and administrators. All mockup links open a static annotated replica of the live page.

Design notes: All pages share a unified Bootstrap 5.3.3 layout with a consistent navbar, light/dark toggle (data-bs-theme), and platform name injection via theme.js. User pages use a neutral navbar; admin pages use a red (bg-danger) navbar with an ADMIN badge.

User Pages

Browse Videos

/public/browse.html Open Mockup ->

The main catalog. Users can search, filter, and sort all available videos, then click a card to open the Watch page.

UI ElementFunction
Navbar brand + linksConsistent navigation; active link highlighted.
Search inputFilters by title, creator name, description.
Sort dropdownNewest / Cheapest / Most expensive / A-Z.
Video cardsClickable -> goes to watch page; shows Ready or Processing badge, plus affiliate badge when the creator has enabled a commission program for that video.
Free badgeGreen text when the video costs $0.
Price displayIn Rupiah using the live USD to IDR rate from /api/kurs.
User barShows logged-in username; shows Sign In link if not logged in.
Theme toggleSwitches light/dark and is saved to localStorage.

User Dashboard

/public/dashboard.html Open Mockup ->

Central hub for content creators: manage profile, upload videos, configure pricing, grant access, and chat with admin or other users.

SectionFunction
Profile & Payout InfoEmail (read-only), WhatsApp, bank account, EVM wallet, preferred chain, and bio for fiat/crypto disbursement.
Change PasswordRequires current password plus new password confirmation. Uses POST /api/change_password.
My Videos & AccessLists uploaded videos with HLS status, inline edit form, and per-video allowlist.
Upload VideoMultipart MP4 upload. Server transcodes to HLS automatically.
Grant Access (Allowlist)Pick a video and enter a username to grant access without payment. Uses POST /api/allow.
ChatUser can open a support thread with admin, search other users, start direct conversations, and send messages from the browser. All messages are stored in the database via /api/chat/*.

Wallet

/public/wallet.html Open Mockup ->

Manage platform balance. Top-up via crypto or bank, withdraw to bank/crypto, transfer to another user.

Tab / ElementFunction
Balance cardShows available balance in USD and IDR equivalent. Refresh button reloads from server.
Deposit / Top Up tabChoose amount and method (x402 crypto / bank / QRIS). Shows payment instructions after submit.
Withdraw tabRequest withdrawal; destination is taken from the Profile page.
Transfer tabSend balance to another user by username with live lookup.
History tabFull transaction log loaded from the wallet API.

Affiliate Program

/public/affiliate.html Open Mockup ->

Creators can configure commission percentage per video, and affiliates can earn from every purchase made through their referral link.

SectionFunction
Summary cardsTotal earnings, monthly earnings, referral count, and referral code.
Earnings tabCommission history: video, buyer, amount, payment method, and status.
Creator Settings tabVideo owners can set commission_pct per paid video and enable or disable the affiliate program.
Get Links tabSelect a paid video to generate a ?ref=username URL and see whether the affiliate program is enabled for that video.

Watch / Payment

/public/watch.html Open Mockup ->

The pay-per-view page. Shows the HLS player if the user has access; otherwise shows a 3-tab payment panel.

ElementFunction
Locked player overlayShown before payment. Displays price in USD and IDR.
Affiliate program infoIf the video has an active affiliate setup, the page shows the commission percentage near the title and in an info banner.
Wallet tabInstant payment from wallet balance.
Crypto (x402) tabInitiates an x402 on-chain payment and confirms automatically.
Bank Transfer tabUser confirms after manual transfer; access is granted after verification.
HLS playerStreams after access is confirmed.
Video descriptionShows creator description below the player if available.

Flow summary: User registers -> browses videos -> watches free or pays (wallet / x402 / bank) -> video unlocks. Creators upload videos, set prices, grant manual access, configure affiliate commission per video, and can chat with admin or other users. Affiliates share referral links and earn according to the configured commission. Admin reviews support chats, fiat payments, approves withdrawals, and configures platform settings.
